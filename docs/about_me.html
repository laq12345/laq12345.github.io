关于我

我的学业、工作与生活

学业

郑州大学

我就读于郑州大学临床医学专业，现如今我已经大四了。这四年时光里我刻苦学习，不曾懈怠。如今我的所有专业课都已经学完了，最终的获得了11/572的成绩，并成功获得了理想学校待录取通知给我的大学时光画了一个圆满的句号。

重症医学

接下来我的中心或许会放在重症医学知识的学习上，我已经看完了研究生教材，但是总觉得写的很难懂，我买了一本实用重症医学参考书，或许可以学一学。

工作

实习

我目前正在郑州大学第二附属医院实习，大概实习一年时间，在这一年时间里希望能够继续进步，继续努力，学习更多临床知识。

爱好

我目前正在学习各种编程知识，我计划在未来的一年时间里持续学习。

Python

Python是我这一年里的学习重点，我打算掌握Python编程的基本技能，能够自主写一些脚本服务于日常工作和生活。

其次是学习Pandas、NumPy，这是我通向数据科学的必经之路，目前Pandas已经学完了，但是真正运用自如还需要坚持练习。NumPy还需要更深入地学习，这是我接下来深入学习机器学习算法的基础。

最后我还会从学习机器学习算法，从Scikit-Learn和PyTorch开始。

R

R语言是我最初接触的编程语言，这是我开启编程知识学习大门的基石，我觉得我最近有点属于学习，不知道我的数据分析和可视化能力退步了多少，接下来我的学习重心或许会放在Seruat、Signic等单细胞数据分析库的学习。

Linux

Linux我觉得我也应该多花费时间学习一下，目前已经租了一台长期的服务器，我的工作重心或许会放在Vim技能训练和Shell脚本的编写上。
